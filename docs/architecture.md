## 3. 模块依赖关系

```
main.py (入口)
    ├── VideoProcessor (核心处理)
    │       ├── PersonDetector (YOLOv8检测)
    │       ├── StateClassifier (MobileNetV3分类)
    │       ├── DrowningTracker (时序追踪)
    │       ├── AlarmManager (报警)
    │       └── EventLogger (日志)
    │               ├── CSV日志
    │               ├── JSONL日志
    │               └── Elasticsearch (可选)
    │
    ├── MainWindow (PyQt5界面)
    │
    └── settings (配置管理)

web/server.py (Web服务)
    ├── FastAPI框架
    ├── 读取events.jsonl
    └── 提供RESTful API

analytics/ (数据分析)
    ├── event_analyzer.py (分析核心)
    ├── daily_report.py (日报)
    └── weekly_report.py (周报)

bigdata/ (大数据组件)
    └── elasticsearch_client.py (ES客户端)

train/ (训练模块)
    ├── prepare_dataset.py (数据准备)
    └── train_classifier.py (模型训练)
```

## 4. 技术栈总览

| 层级 | 技术/框架 | 用途 |
|------|----------|------|
| **深度学习** | PyTorch + YOLOv8 + MobileNetV3 | 目标检测与分类 |
| **GUI** | PyQt5 | 桌面应用程序 |
| **Web服务** | FastAPI + Uvicorn | Web看板API |
| **数据处理** | Pandas + NumPy | 数据分析 |
| **可视化** | Matplotlib + Seaborn | 图表生成 |
| **大数据** | Elasticsearch (可选) | 事件存储与检索 |
| **视频处理** | OpenCV | 视频流处理 |
| **日志** | JSONL + CSV | 结构化日志 |
